# Changelog

Todas as mudanças importantes neste projeto são documentadas neste arquivo.

O formato segue [Keep a Changelog](https://keepachangelog.com/pt-BR/),
e este projeto segue [Semantic Versioning](https://semver.org/pt-BR/).

## [1.0.0] - 2026-04-05

### ✨ Adicionado
- ✅ Página inicial com hero section e parallax effect
- ✅ Design system completo com paleta de cores e tipografia
- ✅ 15+ procedimentos com filtros por categoria
- ✅ Seção de estatísticas com contadores animados
- ✅ Seção de vantagens com 6 cards principais
- ✅ Sistema de gerador de orçamentos com PDF
- ✅ Página de login com autenticação simples
- ✅ Área administrativa para gestão de orçamentos
- ✅ Adicionar/remover procedimentos e pacotes dinamicamente
- ✅ Preview em tempo real de orçamentos
- ✅ Geração de PDF com html2canvas + jsPDF
- ✅ Ícones sociais flutuantes (WhatsApp e Instagram)
- ✅ Sistema de validação de formulários com feedback
- ✅ Animações suaves com CSS e JavaScript
- ✅ Responsividade completa (mobile, tablet, desktop)
- ✅ Seção de depoimentos de clientes
- ✅ Seção de contato com links diretos
- ✅ CTA section para agendamentos
- ✅ Intersection Observer para animações on-scroll
- ✅ Efeitos de glassmorphism nos componentes

### 🎨 Design & UX
- Paleta de cores harmoniosa (Cyan + Orange)
- Animações fluidas com easing functions
- Gradientes e sombras sofisticadas
- Layout moderno e limpeza visual
- Tipografia bem hierarquizada
- Ícone do WhatsApp e Instagram com gradiente

### 🔧 Tecnologia
- HTML5 semântico
- CSS3 com custom properties (variables)
- JavaScript ES6+ vanilla
- CDN para bibliotecas externas (Animate.css, html2canvas, jsPDF)
- Sem dependências build (puro frontend)

### 📱 Responsividade
- Desktop: 1920px+
- Tablet: 768px - 1024px
- Mobile: 375px - 767px

### 📁 Documentação
- README.md completo
- CONTRIBUTING.md para colaboradores
- LICENSE MIT
- .env.example para variáveis de ambiente
- config.example.json para configurações

### 🔒 Segurança
- Validação de inputs no frontend
- Proteção contra XSS
- localStorage para autenticação (dev)
- HTTPS recomendado
